# める配くん連携WordPressプラグイン

Contact Form 7の投稿内容をめる配くんのAPIを実行して読者登録するWordPressプラグインです。

## 機能

- Contact Form 7のフォーム送信時に自動的にめる配くんに読者登録
- 管理画面でのAPI設定（エンドポイントURL、ログインID、パスワード）
- Contact Form 7のフィールドとめる配くんのCSVレイアウトのマッピング機能
- 重複チェック機能
- 接続テスト機能

## インストール

1. プラグインファイルを `/wp-content/plugins/meruhai-integration/` にアップロード
2. WordPress管理画面の「プラグイン」から「める配くん連携」を有効化
3. 「設定」→「める配くん連携」で設定を行う

## 設定手順

1. **API設定**
   - エンドポイントURL: める配くんのAPIエンドポイント（例: https://devel.mgzn.jp/api/apiText.php）
   - ログインID: める配くんのクライアントログインID
   - パスワード: める配くんのログインパスワード

2. **接続テスト**
   - 設定を保存後、「接続テスト」ボタンをクリック
   - 成功すると読者リストが取得され、フィールドマッピング画面が表示されます

3. **フィールドマッピング**
   - Contact Form 7のフィールドとめる配くんのCSVレイアウトをマッピング
   - 「登録日」「未使用」「SMSステータス」はマッピング対象外
   - マッピングを保存

## 動作

Contact Form 7でフォームが送信されると、設定されたマッピングに基づいてCSVデータが作成され、める配くんの読者登録APIが実行されます。

重複するメールアドレスが検出された場合は、エラーログに記録され、登録はスキップされます。

## 必要な環境

- WordPress 5.0以上
- Contact Form 7 プラグイン
- PHP 7.4以上
- cURL拡張機能

## トラブルシューティング

- 接続テストが失敗する場合：エンドポイントURL、ログインID、パスワードを確認
- 読者登録が失敗する場合：WordPressのエラーログを確認
- フィールドマッピングが表示されない場合：Contact Form 7でフォームが作成されているか確認
